# 🛍️ E-Commerce Website
Simple E-commerce website using HTML, CSS and JAVASCRIPT

## UI
![image](https://user-images.githubusercontent.com/105323431/215714209-efbf84ee-6c84-42d0-8108-d413f2a0dd4a.png)

---

A fully responsive front-end e-commerce website built using **HTML, CSS, and JavaScript**.  
This project features a clean UI design, trending products section, client reviews, and a structured homepage layout suitable for modern e-commerce platforms.

---

## 🚀 Features

### 🏠 Homepage
- Stylish hero banner with call-to-action button  
- Smooth navigation and scroll  
- Winter collection highlight  

### 🛒 Trending Products
- Product cards with images, ratings, and price details  
- Wishlist (heart icon)  
- Responsive grid layout  

### ⭐ Client Reviews
- Customer testimonial section  
- Professional layout with image and designation  

### 📰 Update News Section
- Blog-style update cards  
- Publish dates, descriptions, and "Continue Reading" links  

### 📞 Contact Section
- Company address, email, and phone details  
- Social media icons  
- Support, shop, and company links  

### 📱 Responsive Navbar
- Hamburger menu for mobile  
- Search, user, and cart icons

- **CSS3**  
- **JavaScript**  
- **Google Fonts**
